# Changelog

All notable changes to Archexa will be documented in this file.

## [0.1.0] - 2026-03-12

### Added
- Initial release
- Commands: `gist`, `analyze`, `query`, `impact`, `review`, `doctor`, `init`
- Deep mode (`--deep`) with agentic investigation loop
- Multi-language support via Tree-sitter (15+ languages)
- Evidence-based documentation with E# references
- Mermaid and ASCII diagram generation
- Adaptive token budget management
- Evidence caching for fast repeated runs
- Support for OpenAI, Azure OpenAI, OpenRouter, and any OpenAI-compatible API
- Cross-platform binaries (macOS ARM64/x86_64, Linux x86_64/ARM64)
